# Hello World Static Site

A simple static HTML website for testing Coolify deployment.

## Setup Instructions

1. Create a remote Git repository (GitHub, GitLab, etc.)
2. Add the remote and push:
   ```bash
   git remote add origin <your-repo-url>
   git push -u origin master
   ```
3. Use the repository URL when creating the Coolify application

## Files

- `index.html` - Main HTML file with a simple "Hello World" page
